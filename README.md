# Library Workspace


## Run Locally

Clone the project

```bash
  https://github.com/sametcelikbicak/library-workspace.git
```

Go to the library project directory

```bash
  cd library-workspace
```

Install dependencies

```bash
  npm install
```

Go to the test application project directory
```bash
  cd projects/library-test
```

Start the server

```bash
  npm run start
```
